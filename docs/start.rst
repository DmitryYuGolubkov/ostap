Getting started 
===============

The interactive `ostap` approcation can be incked with a simple command 

.. code-block:: bash
   :linenos:

    ostap 

In particular, there exists a :code:`-h` switch, that provides the extensive help on the possible swicthes

.. code-block:: bash
   :linenos:

    ostap -h 
  
On default a large part of modules is loaded,  providing significant 
amount of functinality   available withtout additional imports
